# ChatGPT Team Indonesia 🤖

A modern, fast, and secure website for ChatGPT Team Indonesia, hosted on Cloudflare Pages.

## 🚀 Quick Start

### Local Development

Using Python (already installed on your system):
    python -m http.server 8000

Or using Node.js:
    npx serve .

### Deployment
This site auto-deploys to Cloudflare Pages on every push to main branch.

Live at: https://chatgpt-team.pages.dev

## 📦 Tech Stack
- Cloudflare Pages (Hosting)
- Vanilla HTML/CSS/JS (No framework needed)
- Cloudflare CDN (300+ locations)
